HTML: estrutura básica

📄

HTML é como o esqueleto da página. Ele cria blocos para títulos, parágrafos, listas e links.

Exemplo:

<h1>Bem-vindo</h1> <p>HTML mostra o conteúdo da página.</p> <ul> <li>Item 1</li> <li>Item 2</li> </ul>

Note que o HTML não decide cores ou movimentos. Ele apenas define o que existe na página.

CSS: aparência visual

🎨

CSS pinta o HTML. Ele escolhe cores, fontes, tamanhos e como os elementos se posicionam.

Exemplo:

body { background: #f3f4f6; font-family: Arial, sans-serif; } h1 { color: #1d4ed8; } button { background: #2563eb; color: white; padding: 12px 24px; border-radius: 999px; }

Com CSS, o mesmo conteúdo HTML pode ter um visual muito diferente.

JavaScript: ação e interatividade

⚡

JavaScript adiciona vida ao site. Ele responde a cliques, mexe no conteúdo e cria animações.

Exemplo:

const botao = document.querySelector('button'); botao.addEventListener('click', () => { alert('Você clicou no botão!'); });

Sem JavaScript, a página mostraria apenas conteúdo estático. Com ele, fica dinâmica e interativa.
