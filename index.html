O que cada linguagem faz

As três linguagens principais de um site trabalham juntas:

  • HTML define a estrutura e o conteúdo da página.
  • CSS define o estilo visual, como cores, fontes e layout.
  • JavaScript controla o comportamento e a interatividade.

Veja exemplos ilustrados

1. HTML: Estrutura do site

HTML define os elementos de uma página, como títulos, textos, imagens, links e formulários.

Quando você digita um endereço, o navegador pede os arquivos ao servidor e lê o HTML para montar a página.

2. CSS: Aparência e estilo

CSS controla cores, fontes, tamanhos, espaçamento e a posição dos elementos.

Ele faz a página ficar visualmente agradável e responsiva para diferentes telas.

3. JavaScript: Interatividade

JavaScript adiciona comportamento ao site, como abrir menus, validar formulários e atualizar conteúdo sem recarregar a página.

4. O que acontece quando você acessa um site?

  1. O navegador envia uma solicitação HTTP ao servidor usando o endereço do site.
  2. O servidor responde com arquivos HTML, CSS e JavaScript.
  3. O navegador lê o HTML e constrói a estrutura da página.
  4. O CSS é aplicado para definir o visual da página.
  5. O JavaScript é executado para tornar a página interativa.
